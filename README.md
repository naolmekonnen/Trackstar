# Trackstar

**Train for your next race like a Trackstar.**

A premium running training app combining AI race plans, cinematic run recaps, social motivation, runner identity, and community discovery.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Next.js 16** with App Router
- **React 19** + **TypeScript**
- **Tailwind CSS v4** + **shadcn/ui**
- **Framer Motion** for animations
- **Lucide React** for icons

## App Structure

| Route | Description |
|-------|-------------|
| `/` | Landing page |
| `/auth` | Sign in / Sign up |
| `/onboarding` | Multi-step onboarding flow |
| `/home` | Main dashboard |
| `/plan` | AI training plan |
| `/run` | Run tracking & history |
| `/recap` | Cinematic share cards |
| `/persona` | Runner persona system |
| `/aura` | Aura visualization & runner score |
| `/predictions` | Race time predictions |
| `/social` | Activity feed, friends, challenges |
| `/discover` | Run clubs & race discovery |
| `/profile` | Runner profile |
| `/settings` | App settings & integrations |
